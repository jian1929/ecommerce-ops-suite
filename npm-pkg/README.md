# E-commerce Skills Suite - NPM Package

> AI-powered skills for e-commerce automation, distributed as an NPM package.

## Installation

```bash
npm install -g @auto-company/ecommerce-skills
```

## Quick Start

```bash
# Install skills to OpenClaw workspace
npx @auto-company/ecommerce-skills install-skills

# List installed skills
openclaw skills list | grep ecommerce
```
