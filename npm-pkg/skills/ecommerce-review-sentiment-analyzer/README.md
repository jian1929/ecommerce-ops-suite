# E-Commerce Review Sentiment Analyzer

**AI-powered review analysis for e-commerce businesses**

## What This Skill Does

Automatically analyze customer reviews to:
- Extract positive/negative sentiment patterns
- Identify product improvement opportunities
- Track customer satisfaction over time
- Generate actionable insights

## Quick Use

```
"帮我分析这个亚马逊产品的 reviews"
"Analyze reviews for ASIN B08N5WRWNW"
"我们店铺的好评率和差评主要是什么"
```

## Key Features

- Multi-platform support (Amazon, Shopify, Walmart, eBay, Etsy)
- Theme-based categorization
- Sentiment trend tracking
- Review response templates
- Actionable recommendations

## Installation

```bash
clawhub install ecommerce-review-sentiment-analyzer
```

## Requirements

- OpenClaw installed
- Web scraping capability for review extraction
- Optional: OpenAI API for enhanced sentiment analysis
