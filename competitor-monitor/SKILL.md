# Competitor Monitor

电商竞品监控技能 - 帮你实时掌握竞争对手动态。

## 功能特性

### 1. 价格追踪
- 追踪竞品商品价格变动
- 设置价格变动阈值（默认 5%）
- 降价时立即通知
- 支持多平台：淘宝、京东、拼多多、Shopify

### 2. 评分监控
- 追踪竞品评分变化
- 新增差评时提醒
- 评分骤降告警

### 3. 新品发现
- 自动发现竞品上架新品
- 新品上架通知
- 产品特征分析

## 使用方法

### 基础监控
```
监控 [竞品名称] 的 [商品类别]
频率: [每天/每周/实时]
```

### 价格告警
```
价格变动超过 [X]% 时通知我
告警渠道: [邮件/微信/Discord]
```

### 定期报告
```
每周一发送竞品周报
包含: 价格变动、评分变化、新品列表
```

## 数据来源

- 公开商品页面（遵守 robots.txt）
- 电商平台 API（如有授权）
- 第三方数据平台（如有授权）

## 约束

- 仅监控公开可见信息
- 不采集个人隐私数据
- 不进行恶意爬取
- 遵守各平台使用条款
