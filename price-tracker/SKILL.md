# Price Tracker

商品价格追踪技能 - 帮你找到最佳购买时机。

## 功能特性

### 1. 价格追踪
- 添加任意商品链接追踪
- 支持平台：淘宝、京东、拼多多、亚马逊、eBay
- 价格历史记录永久保存

### 2. 降价提醒
- 设置目标价格，到价通知
- 降价百分比提醒（如降价 20% 时通知）
- 支持邮件、微信、Discord 通知

### 3. 全网比价
- 同一商品全网比价
- 找出最低价平台
- 显示历史最低价

### 4. 价格图表
- 可视化价格历史
- 识别价格周期规律
- 预测最佳购买时机

## 使用方法

### 添加追踪
```
追踪 [商品链接]
目标价格: [金额]
降价 [X]% 时通知
```

### 查看历史
```
显示 [商品名称] 价格历史
时间范围: 最近 [X] 天
```

### 全网比价
```
比较 [商品名称/链接] 全网价格
```

## 约束

- 仅追踪公开价格信息
- 不采集用户行为数据
- 遵守各平台使用条款
