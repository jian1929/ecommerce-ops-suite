# E-Commerce Email Automation

Create automated email sequences that convert leads into customers and retain existing ones.

## When to Use

Use this skill when the user wants to:
- Set up automated email sequences
- Create cart abandonment campaigns
- Build welcome email series
- Design post-purchase follow-up flows
- Re-engage inactive customers
- Write promotional email campaigns

## Supported Platforms

- Klaviyo, Mailchimp, Omnisend, SendGrid, ConvertKit, Shopify Email
- Custom SMTP integrations

## Email Sequence Types

### 1. Welcome Series (3-5 emails)

**Day 0 - Welcome Email:**
```
Subject: Welcome to [Brand]! 🎉
- Brand introduction and values
- What to expect (email frequency)
- Best sellers or popular categories
- First purchase incentive (10% off)
```

**Day 2 - Value Email:**
```
Subject: Here's what makes us different
- Unique selling proposition
- Customer testimonials
- Brand story snippet
- Link to blog or lookbook
```

**Day 5 - Social Proof:**
```
Subject: Join [X] happy customers
- Customer reviews and photos
- User-generated content
- Press mentions or awards
- Trust badges
```

### 2. Cart Abandonment Sequence

**1 Hour After Abandonment:**
```
Subject: Did you forget something? 👀
- Show cart items with images
- Simple CTA: "Complete Your Order"
- Shipping threshold reminder if applicable
```

**24 Hours Later:**
```
Subject: Still thinking it over?
- Address potential objections
- Show product benefits
- Offer assistance (live chat link)
- Include urgency (limited stock)
```

**72 Hours Later:**
```
Subject: Last chance - [X] left in stock
- Final discount offer (5-10% off)
- Urgency and scarcity
- Alternative product suggestion
```

### 3. Post-Purchase Flow

**Order Confirmation:**
```
Subject: Your order #[ORDER] is confirmed! ✓
- Order details
- Estimated delivery
- How to track
- Support contact info
```

**Shipping Notification:**
```
Subject: Your order is on its way! 🚚
- Tracking number and link
- What to expect in packaging
- Care instructions (if applicable)
```

**Delivery Confirmation:**
```
Subject: Your order has arrived! 📦
- Thank you message
- Review request
- How to contact support
- Social sharing incentive
```

**3-7 Days After Delivery:**
```
Subject: We'd love your feedback
- Review request with direct link
- Social media sharing option
- Related product suggestion
```

### 4. Win-Back / Re-Engagement

**Day 30 - Check-In:**
```
Subject: We miss you, [First Name]
- "It's been a while" tone
- Show new products
- Exclusive return offer (15% off)
```

**Day 60 - Last Chance:**
```
Subject: Final email before we say goodbye
- Acknowledge they may not be interested
- One final incentive
- Easy unsubscribe (build trust)
```

## Email Copy Formulas

### AIDA Framework
```
Attention: [Hook - question, stat, or bold statement]
Interest: [Build curiosity, share value]
Desire: [Paint picture of transformed life]
Action: [Clear, single CTA]
```

### PAS for E-Commerce
```
Problem: [Relatable pain point]
Agitate: [Expand on consequences]
Solution: [Your product/service as answer]
```

### Subject Line Formulas
- **Curiosity**: "The [X] thing no one talks about..."
- **Urgency**: "Last day to [action]"
- **Personalization**: "[First Name], your cart is waiting"
- **Benefit**: "Get [X] in [Y] minutes"
- **Social Proof**: "[X] people bought this today"

## Key Metrics to Track

| Metric | Good | Target |
|--------|------|--------|
| Open Rate | 15-20% | 25%+ |
| Click Rate | 2-3% | 5%+ |
| Unsubscribe Rate | <0.5% | <0.2% |
| Conversion Rate | 1-2% | 3%+ |

## Best Practices

1. **Segment First**: Never send the same email to everyone
2. **Mobile-Optimize**: 60%+ of emails read on mobile
3. **Timing Matters**: Test send times (typically 10 AM or 2 PM work best)
4. **Personalize**: Use name, past purchases, browsing history
5. **Preheader Text**: Write compelling preview text (40-50 chars)
6. **Clear CTAs**: One primary CTA per email
7. **Test Everything**: A/B test subject lines, copy, timing
